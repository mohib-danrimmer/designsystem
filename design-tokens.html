Design Tokens

All CSS custom properties defined in shared/tokens.css. Click any token name to copy it. Tokens are globally available — link the stylesheet once and use them anywhere.

1 — Import
<link rel="stylesheet" href="shared/tokens.css" />
2 — Use
background: var(--color-primary); color: var(--text-primary); padding: var(--space-4); border-radius: var(--radius-lg);
3 — Override (mobile surface)
<div class="surface-mobile"> /* --color-primary resolves to teal */ </div>

Use transition-fast for micro-interactions (hover), transition-normal for theme switches and reveals, transition-slow for page-level animations.

TokenValueUse on
